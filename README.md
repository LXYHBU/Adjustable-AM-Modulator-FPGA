# FPGA-AM-Modulator: 基于 FPGA 的可调深度 AM 调幅系统

## 📖 项目简介
[cite_start]本项目实现了一个基于 FPGA 的全数字调幅（AM）系统。系统通过双通道 ADC (AD9238) 分别采集载波与调制波信号 [cite: 312, 313][cite_start]，在 FPGA 内部完成乘法运算与直流偏置叠加，最终通过双通道 DAC (AN9767) 输出 AM 调制信号 [cite: 314]。
[cite_start]项目最大特色是支持**通过外部实体按键，动态、无缝地切换 7 档不同的调制深度 (Modulation Depth)** [cite: 316, 317, 326, 327]。

## ⚙️ 硬件环境
* [cite_start]**ADC 模块:** AD9238 (12-bit, 双通道) 
* [cite_start]**DAC 模块:** AN9767 (14-bit, 双通道) 
* [cite_start]**开发语言:** Verilog HDL 

## 🗂️ 核心模块解析

* **`ADDA.v` (顶层模块)**
    * [cite_start]整合全链路系统，例化 PLL 时钟分频（生成 65MHz 工作时钟）[cite: 311]。
    * [cite_start]连接底层外设驱动与上层 AM 调制算法 。
* **`AM.v` (AM 调制核心算法)**
    * [cite_start]包含核心调制逻辑：对调制波施加缩放系数（调制度控制），并叠加直流偏置分量（`A=2047`）[cite: 325, 328, 330]。
    * [cite_start]根据按键输入的 `depth` 状态（0~6），动态调整缩放比例系数 [cite: 326, 327]。
* **`v_ad.v` & `v_da.v` (数模转换驱动)**
    * [cite_start]**AD端:** 完成 12 位双通道数据的高速锁存打拍同步 [cite: 351, 352, 354]。
    * [cite_start]**DA端:** 包含针对 24 位乘法结果的截断处理（右移 9 位）、四舍五入补偿以及严格的防溢出限幅逻辑（限制在 14 位有符号数边界内），保障输出波形的平滑无毛刺 [cite: 356, 358, 359, 360]。
* **`key_filter.v` (高可靠性按键消抖)**
    * 基于 20ms 计时器的标准状态机设计（IDLE -> P_FILTER -> WAIT_R -> R_FILTER），有效滤除机械按键抖动，输出纯净的单脉冲触发信号 [cite: 334, 344]。

## 🛠️ 使用说明与注意事项
1.  **IP 核依赖:** 本工程的 `AM.v` 模块调用了 Vivado/ISE 的乘法器 IP 核（`mult_gen_0`, `mult_gen_1`）。克隆项目后，请在你的 EDA 平台中重新生成对应位宽的乘法器 IP。
2.  [cite_start]**调制度切换:** 系统默认开机调制度档位为 0 [cite: 316][cite_start]。每按下一次外部 `Key`，调制度递增，到达第 6 档后循环归零 [cite: 317]。

## 📊 仿真测试
[cite_start]内置 `ADDA_tb.v` 仿真激励文件，通过模拟不断递增的 12 位 AD 输入数据 [cite: 322][cite_start]，可直接在时序波形图中验证乘法调制及限幅输出逻辑的正确性 [cite: 319]。
